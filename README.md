This is a project to build a clone of UnTappd.com but for Wine.

## Versions:
* Ruby 2.4.1
* Rails 5.1.2
* PostgreSQL

## To Test this out yourself
1. Clone this repo
2. `cd uncorked`
3. `bundle`
4. `rake db:setup`
5. run `rails s`
6. open `http://localhost:3000`
7. login as a user with:
 * Username: / Password:
8. login as a venue owner with:
 * Username: / Password:
9. login as a platform admin with:
